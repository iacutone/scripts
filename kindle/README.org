Using [[https://github.com/honza/kindle-highlight-parser][this]] Go library to import Kindle text clippings as org files.

_Future Reference_
`./kindle-highlight-parser-0.4.0 /Volumes/Kindle/documents/My\ Clippings.txt -f ~/Dropbox/orgfiles/kindle-highlights.org -o org`
